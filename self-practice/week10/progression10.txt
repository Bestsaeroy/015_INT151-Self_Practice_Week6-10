=============================
Progression : loadQuotes()
=============================
1) อ่านข้อมูลจาก localStorage
   - ใช้ localStorage.getItem("quotes") เพื่อดึงข้อมูลเก็บถาวรที่เคยถูกบันทึกไว้
2) ตรวจสอบว่ามีข้อมูลหรือไม่
   - ถ้า localStorage ยังว่าง ให้ปล่อย quotes เป็น array ว่าง
   - ถ้ามีข้อมูล เดินไปขั้นตอนถัดไป
3) แปลงข้อมูลจาก JSON string → Object/Array
   - ใช้ JSON.parse() เพื่อแปลงข้อมูลกลับมาเป็น array ที่ใช้งานได้
4) นำข้อมูลมากำหนดเป็นค่าในตัวแปร quotes
   - ตอนนี้โปรแกรมมีข้อมูลพร้อมใช้งานเหมือนก่อน refresh
5) (หน้าที่ต่อเนื่อง) เรียก renderQuotes()
   - เพื่อแสดงข้อมูลที่โหลดขึ้นมาบน UI

=============================
Progression : renderQuotes()
=============================
1) เคลียร์ UI เดิม
   - ตั้งค่า quoteList.innerHTML = "" เพื่อล้างข้อมูลเก่าออกจากหน้าจอ
2) วนลูปข้อมูลใน quotes
   - ใช้ quotes.forEach() เพื่อเตรียมสร้าง UI ให้แต่ละรายการ
3) สร้าง element ของ quote
   - เรียกฟังก์ชัน addQuoteToDOM(q) เพื่อสร้าง <div> ของแต่ละ quote
4) ประกอบ content + author + ปุ่ม Edit/Delete
   - เพื่อให้ผู้ใช้สามารถจัดการ quote ได้
5) แสดงผลบนหน้าเว็บ
   - ใช้ appendChild() เพื่อเพิ่ม quote แต่ละตัวในหน้า UI
