Progression
1. เลือกกล่องข้อความ
   - ใช้ document.getElementById("box") เพื่อเลือก <div id="box">
   - เก็บไว้ในตัวแปร box เพื่อเรียกใช้ภายหลัง

2. เลือกปุ่มทั้งหมดในหน้า
   - ใช้ document.querySelectorAll("button") เพื่อเลือกปุ่มทั้ง 3 ปุ่ม
   - ได้ผลลัพธ์เป็น NodeList และเก็บไว้ในตัวแปร buttons

3. วนลูปปุ่มทุกปุ่ม
   - ใช้ buttons.forEach(...) เพื่อเตรียมสั่งให้ปุ่มทุกปุ่ม "คลิกแล้วทำงานได้"

4. ผูก Event คลิกกับปุ่มแต่ละอัน
   - ใช้ btn.addEventListener("click", () => { ... })
   - เมื่อปุ่มถูกคลิก โค้ดที่อยู่ภายใน callback function จะทำงานทันที

5. อ่านค่าจาก data-color ของปุ่มที่ถูกคลิก
   - ใช้ btn.dataset.color เพื่ออ่านค่า เช่น "red"

6. เปลี่ยนสีพื้นหลังของกล่อง
   - นำค่าที่อ่านได้ไปกำหนดให้ box.style.backgroundColor = color

7. อัปเดตข้อความเพื่อแสดงผลลัพธ์บนหน้าจอ
   - ใช้ box.textContent = color.toUpperCase() เพื่อเปลี่ยนข้อความภายในกล่อง
   - ทำให้ผู้ใช้เห็นว่าสีที่เลือกคือสีอะไร


## index1 && main1 ##
ในแบบฝึกนี้ เราจะฝึกใช้ DOM Manipulation เพื่อเปลี่ยน ขนาดตัวอักษร (font-size) ภายในกล่องข้อความ โดยอ้างอิงค่าจาก dataset ของปุ่มที่ผู้ใช้กด
โดยหน้าเว็บจะมีองค์ประกอบดังนี้:
    กล่องข้อความ <div id="textBox"> ที่แสดงคำว่า "Hello DOM"
    ปุ่ม 3 ปุ่ม คือ เล็ก, กลาง, ใหญ่ ซึ่งแต่ละปุ่มมีแอตทริบิวต์ data-size เก็บค่าขนาดตัวอักษรที่ต้องการ เช่น 16px, 24px, 40px
    เมื่อผู้ใช้คลิกปุ่มใดปุ่มหนึ่ง ขนาดของตัวอักษรในกล่องจะเปลี่ยนตามค่าที่กำหนดใน data-size และข้อความในกล่องจะอัปเดตเพื่อแสดงผลลัพธ์ เช่น “Font Size : 24px”

1. เลือกกล่องข้อความ
ใช้ getElementById("textBox") เพื่อเลือก <div> แล้วเก็บไว้ในตัวแปร box
2. เลือกปุ่มทั้งหมดในหน้า
ใช้ querySelectorAll("button") เพื่อเลือกปุ่มทั้ง 3 ปุ่ม และเก็บเป็น NodeList
3. วนลูปปุ่มทุกปุ่ม
ใช้ forEach เพื่อติดเหตุการณ์ให้ปุ่มทุกปุ่มสามารถคลิกแล้วทำงานได้
4. ผูก Event คลิกกับปุ่มแต่ละอัน
ใช้ addEventListener("click") เพื่อให้โค้ดภายในทำงานเมื่อถูกคลิก
5. อ่านค่าจาก data-size ของปุ่มที่ถูกคลิก
ใช้ btn.dataset.size เพื่อดึงค่าขนาด เช่น "24px"
6. เปลี่ยนขนาดตัวอักษรในกล่องข้อความ
นำค่าที่ได้ไปกำหนด box.style.fontSize = size
7. อัปเดตข้อความเพื่อแสดงผลลัพธ์บนหน้าจอ
ใช้ box.textContent = "Font Size : 24px" เพื่อให้ผู้ใช้รู้ว่าตอนนี้ใช้ขนาดเท่าไหร่
