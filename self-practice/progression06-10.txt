## Progression Report (Week 06 – Week 10) ## 

Week 06 – Functional Programming
สัปดาห์นี้เป็นพื้นฐานสำคัญของ Functional Programming โดยเน้นมุมมองว่าฟังก์ชันคือข้อมูล (First-class Function) และสามารถส่งเป็นอาร์กิวเมนต์ให้ฟังก์ชันอื่น 
หรือถูกส่งกลับจากฟังก์ชันได้ (Higher-order Function) ทำให้โค้ดมีความยืดหยุ่นและนำกลับมาใช้ซ้ำง่ายขึ้น เราฝึกใช้วิธีเขียนแบบ Declarative ผ่าน map, filter, reduce 
เพื่อลดการใช้ loop แบบ imperative พร้อมทั้งเรียนรู้แนวคิด Immutability, Pure Function และ Function Composition เพื่อให้โค้ดอ่านง่าย ทดสอบง่าย และคาดเดาการทำงานได้ชัดเจน
ผลลัพธ์ของสัปดาห์นี้คือเข้าใจแนวคิดการเขียนโค้ดให้เป็นระบบ ลด side-effect และสร้างฟังก์ชันที่สามารถนำไปประกอบร่วมกันได้อย่างยืดหยุ่น

Week 07 – DOM Manipulation (Create / Append / Text / Attribute)
สัปดาห์นี้เริ่มเข้าสู่การโต้ตอบกับหน้าเว็บจริงผ่าน DOM โดยฝึกสร้าง element แบบ dynamic ด้วย document.createElement() และแสดงผลบนหน้าเว็บด้วย appendChild() พร้อมทั้งทดลองความแตกต่างระหว่าง 
innerHTML, innerText และ textContent เพื่อเลือกใช้ให้เหมาะสมกับสถานการณ์ นอกจากนี้ยังได้ฝึกใช้ setAttribute() เพื่อกำหนดข้อมูลและคุณสมบัติให้ element ที่สร้างขึ้น ทำให้ควบคุม UI ด้วย JavaScript 
ได้อย่างเป็นระบบและปรับเปลี่ยนได้ตามข้อมูลหรือเงื่อนไขของโปรแกรม

Week 08 – Event & Data Attribute Practice
สัปดาห์นี้เน้นการตอบสนองต่อการกระทำของผู้ใช้ (Event) และการอ้างอิงข้อมูลจาก dataset ของปุ่มในหน้าเว็บ เช่น data-size, data-color เพื่อเปลี่ยน style หรือข้อความแบบ real-time
สิ่งที่ฝึกคือการวนลูปปุ่ม, ผูก event click, อ่านค่า dataset และอัปเดต DOM ทันทีที่เกิด action ทำให้เข้าใจว่าการโต้ตอบกับผู้ใช้สามารถควบคุมด้วย JavaScript ได้อย่างไร และเริ่มเชื่อมโยง DOM + Event + Logic เข้าด้วยกันอย่างสมบูรณ์

Week 09 – Mini Project (Quote App: Add / Edit / Delete / Random)
สัปดาห์นี้นำความรู้ที่สะสมมาต่อยอดเป็น mini project — ระบบจัดการ Quote ที่สามารถเพิ่ม, แก้ไข, ลบ และสุ่มข้อมูลได้ โดยมี flow การทำงานแบบครบวงจร ตั้งแต่การเลือก DOM, ผูก event, จัดการข้อมูลใน array 
ไปจนถึงอัปเดต UI แบบ real-time โดยไม่ต้อง refresh หน้าเว็บผลที่ได้จากสัปดาห์นี้คือเข้าใจการทำงานเป็น flow, มองภาพการเชื่อมระหว่าง logic + DOM + event 

Week 10 – LocalStorage & Rendering UI
สัปดาห์นี้เป็นการเพิ่มความสมบูรณ์ให้โปรเจกต์เดิม โดยเก็บข้อมูล Quote ลงใน localStorage เพื่อให้ข้อมูลคงอยู่แม้ refresh หน้าเว็บ พร้อมฝึกกระบวนการ loadQuotes() และ renderQuotes()
การทำงานคือดึงข้อมูลจาก localStorage → แปลงกลับเป็น array → render UI ใหม่ทุกครั้งที่เปิดหน้าเว็บ ผลลัพธ์คือโปรเจกต์เริ่มมีความเป็น “แอปจริง” มากขึ้น เพราะมีการเก็บ state ถาวร

สรุปภาพรวม
สิ่งที่พัฒนาขึ้นชัดเจน:
- เขียนโค้ดที่เป็นระบบมากขึ้น
- เข้าใจการโต้ตอบกับ DOM และ Event
- ใช้ JavaScript ทำ UI ที่ dynamic ได้
- แยกไฟล์และแยกหน้าที่โค้ดชัดเจน
- สร้าง mini project ที่มี state, event, และ data flow ครบวงจร
